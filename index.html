Grayjay Plugins

These are plugins that I have developed that can be installed on GrayJay.

To install a plugin, you need to have the GrayJay app installed on your device. You can find it in grayjay.app.

Once you have GrayJay installed, click the "Open in Grayjay" button below each plugin to install it.
